# ToDo-using-only-JS

A simple and intuitive To-Do List application designed to help users manage tasks efficiently. This project demonstrates the core functionality of adding, updating, and deleting daily tasks with a clean and responsive user interface.

## 🚀 Features

- ✅ Add new tasks with ease  
- 📝 Edit existing tasks  
- ❌ Delete tasks once completed or unnecessary  
- 💾 Real-time task updates *(e.g., using local storage or backend integration)*  
- 🎨 Responsive and clean UI compatible with all screen sizes  

## 🛠️ Tech Stack

- HTML5   
- JavaScript
